# EasyToLearn
EasyToLearn is a web-based Learning Management System (LMS) developed to simplify the teaching and learning process. It enables students, teachers, and administrators to access and manage educational resources through an easy-to-use interface.
## Features
- Student Login
- Teacher Login
- Admin Login
- User-friendly Interface
- Easy Navigation
## Technologies Used
- PHP
- MySQL
- HTML
- CSS
